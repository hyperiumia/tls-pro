# Hyperium TLS-Pro

### Deep TLS/SSL Security Assessment & Certificate Analysis

`v2.0.0` | Patricio Tirado | Hyperium

---

## TLS configuration is the front door of every secure service. Misconfigured cipher suites, expired certificates, deprecated protocols, and weak key exchanges are found in production systems every day.

---

## What It Does

A dedicated TLS/SSL assessment engine that performs deep analysis of server TLS configuration: cipher suite inventory, certificate chain validation, protocol version audit, and quantum vulnerability assessment.

---

## Capabilities

| Capability | Description |
|---|---|
| **Cipher Suite Inventory** | Complete enumeration of supported cipher suites with strength classification |
| **Certificate Chain Analysis** | Full chain validation, expiry checking, and key strength assessment |
| **Protocol Version Audit** | Detection of deprecated TLS 1.0/1.1 and SSLv3 support |
| **Key Exchange Analysis** | ECDHE, DHE, RSA key exchange classification with quantum vulnerability flags |
| **HSTS & OCSP Validation** | HTTP Strict Transport Security and certificate revocation checking |
| **Bulk Scanning** | Concurrent multi-server scanning with configurable parallelism |
